API Server demo

KeyTakeAway:

- Start local services
- Implment and demo Unit tet with vitest
- Implment and demo Contract testing with pact

### Start

Start express servers

```
npm start
```

```
http://localhost:3000/users/1
```

All test

```
npm run test:all
```

Unit test with Vitest

```
npm run test:unit
```

Contract test with PactJS

```
npm run test:contract
```
